Etch A Sketch
Hecho con: JavaScript, CSS, HTML
Características: Modo color, Selector de color, Modo arcoíris, Borrador, Botón de limpiar, Selector de tamaño de cuadrícula.

Modo color: La función básica para crear tu propio pixel art, como un lápiz digital.
Selector de color: Permite elegir cualquier color usando RGB, HSL, HEX, una paleta o cuenta gotas.
Modo arcoíris: Cada píxel cambia a un color aleatorio al pintar, como un arcoíris.
Borrador: Herramienta simple para eliminar colores del lienzo.
Botón de limpiar: Borra todo el contenido del lienzo y permite empezar de nuevo.
Selector de tamaño de cuadrícula: Permite ajustar el tamaño del lienzo desde 1x1 hasta 100x100 píxeles.

Calculator
Hecho con: JavaScript, CSS, HTML
Características: Soporta todas las operaciones básicas (dividir, multiplicar, residuo, sumar, restar). Incluye opciones para eliminar un solo número o limpiar toda la pantalla. También permite trabajar con números decimales en todas las operaciones.

Teclado: Compatible con entradas desde el teclado físico para todos los botones.
Teclas rápidas:
Esc: AC/Limpiar todo.
c / C / Backspace: Borrar un solo número.
Las demás teclas funcionan igual que en la calculadora visual.
